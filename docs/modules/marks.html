Module marks

This is the marks module.

It massively improves the performance while rendering an entity (highlighting it) with caching

Info:

  • Author: Alf21

Functions

Add (ents, col) Adds entities into the entities list that should be rendered with a specific color

Functions

Add (ents, col)
Adds entities into the entities list that should be rendered with a specific color

Parameters:

  • ents table list of entities that should be added
  • col Color the color the added entities should get rendered
Clear ()
Clearing the cached entity list
Remove (ents)
Removes entities from the entities list

Parameters:

  • ents table list of entities that should get removed
Set (ents, col)
Sets entities of the entities list that based on a specific color. All the other previously inserted entities with the same color will get removed

Parameters:

  • ents table list of entities that should be set
  • col Color the color the added entities should get rendered

Usage:

    marks.Set({}, COLOR_WHITE) -- this will clear all entities rendered in white
