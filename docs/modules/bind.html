Register (name, onPressedFunc, onReleasedFunc[, dontShowOrCategory][, settingsLabel][, defaultKey])
Register a function to run when the button for a specific binding is pressed. This will also add the binding to the BindingsCategories with the default category, otherwise set dontShowOrCategory to true. If you wish to set a custom category name then set dontShowOrCategory to the category name and optionally also set the label you wish to be displayed. You can set a default key as well, so players don't need to bind manually if they don't want to.

Parameters:

  • name string '
  • onPressedFunc func '
  • onReleasedFunc func '
  • dontShowOrCategory string or bool ' (optional)
  • settingsLabel string ' (optional)
  • defaultKey number ' (optional)
